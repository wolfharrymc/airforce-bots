# 🐺 WOLFBOT | SWARM TERMINAL

Advanced Minecraft bot management system with tactical swarm controls and automated response features.

## Features
- **Swarm Control**: Manage multiple bots simultaneously from a unified terminal.
- **Tactical Movement**: Real-time directional control including jumping.
- **Smart Responses**: Whitelisted `!come` and `!tpa` commands with automated server responses.
- **Savage Defense**: Automated roasts for non-whitelisted players.

## Deployment on Hugging Face
This project is configured to run on Hugging Face Spaces using Docker. It exposes port 7860 for the Swarm Terminal UI.
